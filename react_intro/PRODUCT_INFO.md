# FocusFlow - Personalized Productivity System

**Produktnamn:** FocusFlow  
**Kurs:** React, TypeScript och React Native (CHAS Academy)  
**Projekt:** Boiler Room Projekt  
**Leverans:** Kursvecka 12

## 📋 Projektbeskrivning

FocusFlow är ett produktivitetsverktyg som hjälper användare att optimera sin arbetsdag genom:

- ⏱️ **Tidsspårning**: Spåra arbete med precision
- 📊 **Energiloggning**: Visualisera energinivåer över dagen
- 🎯 **Smarta rekommendationer**: Få förslag på optimala arbetsblock
- ⏸️ **Paushantering**: Ta pauser vid rätt tillfällen
- 🔄 **Fokuslägen**: Växla mellan Deep Work, Möte och Paus

Applikationen levereras som **webbapplikation (React)** och **mobilapplikation (React Native)**, med delad logik mellan plattformarna.

## 🛠️ Tech Stack

- **React 19+** med Vite
- **TypeScript** för typsäkerhet
- **Context API** & **Hooks** för state-hantering
- **React Router** för navigation
- **CSS Modules** för styling
- **Jest & React Testing Library** för testning
- **React Native + Expo** för mobilapp
